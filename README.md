# e-commerce-manual-testing-project

## Overview

This repository contains a Manual QA Testing Portfolio Project for a women's volleyball apparel e-commerce website.

## Project Objective

Validate the end-to-end purchase flow of the Official Women's Volleyball Jersey, ensuring the user can successfully access the product page, select the desired size, add the product to the cart, complete checkout, and receive order confirmation without errors.
